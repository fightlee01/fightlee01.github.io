                                  # 数据结构

                                  # 绪论

                                  • 数据结构3要素：逻辑结构，存储结构（物理结构）和数据的运算
                                  • 数据结构的五个特征：有穷性、确定性、可行性、输入和输出
                                  • 数据类型：原子类型、结构类型和抽象数据类型
                                  • 存储结构主要由：顺序存储、链式存储、索引存储和散列存储（hash存储）

                                  # 线性表

                                  # 线性表的顺序表示——顺序表

                                  线性表的顺序存储又称顺序表，用一组地址连续的存储单元依次存储线性表中的数据元素，从而使逻辑上相邻的两个元素在物理位置上也相邻。顺序表的特点是可以随机存取，每个节点只存储数据元素，存储密度高，但是元素的插入和删除需要移动大量元素。

                                  # 线性表的链表表示——单链表

                                  线性表的链表表示不需要地址连续的存储单元，逻辑上相邻的元素在物理位置上不一定相邻，因此插入和删除操作不需要移动元素，但是也丢失了随机存取的优点。对于链表中的每个节点，不仅需要存储数据元素的自身信息，还需要存储下一个节点的位置信息。
                                  头节点和头指针的区分：头节点指带头节点的链表中第一个节点，头指针指链表的第一个头节点。加入头节点的优点是不用对头节点作特殊处理。

                                  # 头插法

                                  头插法建立链表时读入的数据顺序与生成的链表中的数据的顺序相反。

                                  #include<iostream>

using namespace std;

typedef struct LNode {
    int data;
    struct Lnode *next;
}LNode, *LinkList;

int main() {
    LNode* s;
    int x;
    LinkList L = (LinkList) std::malloc(sizeof(LNode));
    // L->next = nullptr;
    cin>>x;
    while(x != 999) {
        s = (LNode*) std::malloc(sizeof(LNode));
        s->data = x;
        s->next = L->next;
        L->next = reinterpret_cast<struct Lnode *>(s);
        cin>>x;
    }
    while(L) {
        cout<<L->data<<endl;
        L = reinterpret_cast<LinkList>(L->next);
    }
    return 0;
}

                                  # 尾插法

                                  将新增节点添加到链表的尾部，实现中增加一个尾指针r。

                                  #include<iostream>

using namespace std;

typedef struct LNode {
    int data;
    struct Lnode *next;
}LNode, *LinkList;

int main() {
    LNode* s;
    int x;
    LinkList L = (LinkList) std::malloc(sizeof(LNode));
    LNode* r = L;
    // L->next = nullptr;
    cin>>x;
    while(x != 99) {
        s = (LNode*) std::malloc(sizeof(LNode));
        s->data = x;
        // s->next = L->next;
        // L->next = reinterpret_cast<struct Lnode *>(s);
        r->next = reinterpret_cast<struct Lnode *>(s);
        r = s;
        cin>>x;
    }
    r->next = nullptr;
    while(L) {
        cout<<L->data<<endl;
        L = reinterpret_cast<LinkList>(L->next);
    }
    return 0;
}

                                  # 线性表的链表表示——双链表

                                  双链表在单链表的基础上增加了一个指向其前驱的prior指针。

                                  # 双链表插入

                                  s->next = p->next;
p->next->prior = s;
s->prior = p;
p->next = s;

                                  # 双链表删除

                                  p->next = q->next;
q->next->prior = p;
free(q);

                                  3. 高度为h的m叉树至多有(mh−1)/(m−1)(m^h-1)/(m-1)(mh−1)/(m−1)个节点。

                                  # 二叉树性质

                                  • 非空二叉树的叶子结点等于度为2的结点数加1
                                  • 非空二叉树上第k层至多有2k−12^{k-1}2k−1个结点
                                  • 高度为n的二叉树之多有2n−12^{n-1}2n−1个结点
                                  • 对完全二叉树丛上到下、从左到右的顺序依次编号1, 2, ..., n，则有以下关系:
                                  1. 当i>1i>1i>1时，结点i的双亲编号为i/2取整，即当i为偶数时，其双亲编号为i/2，它是双亲的左孩子，当i为奇数时，其双亲编号为(i-1)/2，它是双亲的右孩子
                                  2. 当2i<=n时，结点i的左孩子编号为2i，否则无左孩子
                                  3. 当2i+1<=n时，结点i的右孩子编号为2i+1，否则无右孩子
                                  4. 结点i所在的层次为log2ilog_2ilog2​i+1
                                  • 具有n个结点的完全二叉树的高度为log2(n+1)log_2(n+1)log2​(n+1)（向上取整）或log2n+1log_2n+1log2​n+1（向下取整）

                                  # 图的基本概念

                                  1. 简单图：如果一个图G满足不存在重复边且不存在顶点到自身的边，那么称图G为简单图。
                                  2. 完全图：对于有n个顶点的无向图，若有n(n−1)/2n(n-1)/2n(n−1)/2条边，则称该图为完全图，在完全图中任意两个顶点之间都存在边；对于有向图，有n(n−1)n(n-1)n(n−1)条边的有向图是完全有向图。
                                  3. 子图：设有两个图G=(V,E)G=(V, E)G=(V,E)和G′=(V′,E′)G'=(V', E')G′=(V′,E′)，若V′V'V′是VVV的子集，且E′E'E′是EEE的子集，则称G′G'G′是GGG的子图。若有满足V(G′)=V(G)V(G')=V(G)V(G′)=V(G)的子图G′G'G′，则称其为G的生成子图。
                                  4. 在无向图中，若从顶点v到顶点w有路径存在，则称v和w是连通的。若图G中任意两个顶点都是连通的，则称图G为连通图。无向图中极大极大连通子图成为连通分量。
                                  5. 在有向图中，如果有一对顶点v和w，从v到w和从w到v之间都有路径，则称这两个顶点是强连通的。若图中任何一对顶点都是强连通的，则称此图为强连通图。
                                  6. 连通图的生成树是包含图中全部顶点的一个极小连通子图。若图中顶点数为n，则它的生成树含有n-1条边。包含图中全部顶点的极小连通子图，只有生成树满足这个极小条件，对生成树而言，若砍去它的一条边，则会变成非连通图，若加上一条边则会形成一个回路。在非连通图中， 连通分量的生成树构成了非连通图的生成森林。
