                              # 注入点判断

                              id=1 and 1=1返回正常，id=1 and 1=2返回错误。此时可能存在sql注入。

                              猜解列名数量（字段数）：order by x，页面报错就是临界值，即列名数。

                              信息收集：

                              数据库版本数据名字数据库用户操作系统版本
                              version()database()user()@@version_compile_os

                              在mysql 5.0以上版本中，自带一个名为information_schema的数据库，该数据库记录了本数据库中所有的数据库名、表名和列名。information_schema.tables记录所有表名的表，information_schema.columns记录所有列名的表。

                              # 文件读取和写入

                              文件读取

                              select load_file('xxx');

                              文件写入

                              select 'x' into outfile 'xxx';

                              # 注入技巧

                              1.明确参数的类型：数字、字符等，搜索框处的注入一般是字符
                              2.明确请求方法：get、post、request、cookie等

                              # SQL注入工具

                              sqlmap、NoSQLAttack、Pangolin

                              # 其他数据库注入

                              1、access：没有数据库的概念，列名只能暴力猜解。

                              # MySQL盲注常用函数

                              length()、substr(str, start_idx, length)、left(str, length)、right(str, length)、regexp()、ord()、mid()
