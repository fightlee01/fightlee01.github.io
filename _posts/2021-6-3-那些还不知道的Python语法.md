# 那些还不知道的Python语法
## __getitem__方法
__getitem__方法是一个魔改方法，它可以改变类的实例访问的方式。例如demo为一个类的实例对象，当使用demo[item]时调用该方法。
```python
class Demo:
    def __init__(self, name, age):
        self.name = name
        self.age = age
    
    def __getitem__(self, item):
        return item + self.name + self.age


demo = Demo('lee', 22)
print(demo[123])#123
print(demo['fight'])#fight
```
